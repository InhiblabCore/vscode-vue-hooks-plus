# VueHooks Plus

`VueHooks Plus` 在 `Vscode`中的多功能预览插件。

## 功能

- 支持官方文档在线查看
- 支持修改 URL 定位到你喜欢的网站。

## Command

`command` + `shift` + `p` 调出输入框，输入 `vue hooks plus` 进行操作。
